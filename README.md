## ⚙️ Adding a New Project

Open `harsha_portfolio.html`, find the `<!-- PROJECTS -->` comment, and add a new card inside the `.projects-grid` div:

```html
<div class="project-card">
  <div class="project-num">04</div>
  <div class="project-title">Your Project Title</div>
  <div class="project-desc">Short description of what it does.</div>
  <div class="project-tech">
    <span class="tech-pill">Python</span>
    <span class="tech-pill">Django</span>
  </div>
</div>
```

## 🚢 Deployment

### Option 1 — GitHub Pages (Free)
1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`

### Option 2 — Netlify (Free, Easiest)
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag and drop your entire project folder
3. Done — instant live URL!

> ⚠️ Make sure `Harsha_V.pdf` and `passport.jpg` are in the **same folder** as the HTML file for the resume download and photo to work correctly.
